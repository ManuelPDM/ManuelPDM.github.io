Raven's Progressive Matrix Project

This project is an exploration of cognitive patterns and problem-solving algorithms inspired by Raven's Progressive Matrices. It delves into pattern recognition and innovative AI-driven reasoning techniques.

Project Overview

The project uses advanced image processing techniques to analyze and solve visual puzzles. At its core, the system:

  • Image Processing: Uses OpenCV and Pillow to convert images into grayscale, apply thresholding, and extract key contours.
  • Feature Extraction: Extracts geometric information from each image, such as the number of edges, shape orientation, area, and fill state.
  • Pattern Recognition: Compares patterns between different sections of the matrix. By evaluating changes in shape, movement, scaling, and rotation, the algorithm infers the underlying rules.
  • Overlay Techniques: Employs image overlay and pixel-difference methods to capture subtle variations and validate the deduced patterns.
  • Decision Making: Aggregates various similarity scores to select the best answer, showcasing a multi-faceted approach to problem solving.

The design intentionally abstracts complex operations and data flows, ensuring that while the approach is innovative, the implementation details remain non-trivial to replicate. This highlights both technical proficiency and creative problem-solving skills.
